# Brandweer Sassenheim
Public website about the Fire Department of Sassenheim, Netherlands.

## How to run
- `git clone git@github.com:Vugario/BrandweerSassenheim.git && cd BrandweerSassenheim`
- `make provision`
- `make up`
- Visit http://0.0.0.0:9000/
